版本二：輕鬆慢活版概覽

日期 時間 行程內容 美食/餐飲 備註

下午/晚 voco 頂樓泳池/酒吧放鬆或美溪海灘漫遊 晚餐：All Seasons Buffet Da Nang 或 POSEIDON BUFFET 海鮮自助餐 建議提前預訂晚餐。

4/4 上午 voco Check-out → Four Points **寄放大型行李** → 前往會安。 午餐：會安古鎮周邊高樓麵 (Cao Lầu) 或會安雞飯 重要移動日：輕裝前往會安。
下午/晚 La Charm Hoi An Check-in → **會安古鎮 (Hội An)** 漫步、參觀、拍照。 晚餐：會安河畔餐廳 (燈籠、特色點心) 體驗會安著名的燈籠夜景與放水燈。
晚上
4/5 上午 La Charm 早餐 → **山茶半島 (Son Tra Peninsula)**：靈姑寺 → Four Points Check-in。 午餐：韓市場 (Han Market) 或 崑市場 (Cồn Market) 市場採購請注意物品安全。
下午 峴港粉紅大教堂 (Da Nang Cathedral) → 韓市場/逛街購物。 晚餐：龍橋附近餐廳、海鮮大排檔
晚上 **SPA投產建議** (如：Queen Spa、Trees Spa、Nguon Spa 等) 建議提前預約。
深夜 龍橋噴火表演 (約晚上9點 - 週末限定) 週末限定，需確認表演表演時間。
4/6 上午 行程：**佔婆博物館 (Cham Museum)** 了解當地歷史文化或美溪海灘邊漫遊放鬆。 午餐：海鮮餐廳或越式法包
下午 逛街購物 (如：Vincom Plaza 或 Lotte Mart)，購買伴手禮。 晚餐：自選喜歡的當地美食或網紅餐廳
晚上 第二次 SPA 投產 (視體力決定)，或於海灘散步。
深夜 飯店頂樓閣樓 lounge 喝酒。
4/7 上午 飯店頂層遊泳池悠閒時光、最後買買、收拾行李。 午餐：附近的飯店或喜歡的餐廳
16:00 前往機場有交通搭乘與報到時間。 18:45 飛機離開 留足充裕時間前往機場。

版本一：巴拿山挑戰版概覽

日期 時間 行程內容 美食/餐飲 備註
4/2 19:00後 voco Check-in → 於飯店周邊或美溪沙灘 (My Khe Beach) 輕鬆晚餐。 晚餐：美溪沙灘在地美食 (如：越式法棍 Bánh Mì 或 Phở) Day 2 體力負荷大，Day 1 需充分休息。
4/3 上午 **巴拿山 (Ba Na Hills) 一日遊：** 佛手金橋、法國村、夢想樂園。 午餐：巴拿山餐廳或美食廣場 (建議提前預訂套票) 建議穿著輕便舒適的鞋子。務必提早出發，避開人潮。
下午/晚 搭纜車下山 → Grab 返回市區。 晚餐：All Seasons Buffet Da Nang 或 POSEIDON BUFFET 海鮮自助餐 建議提前預訂海鮮自助餐位，犒勞一天的辛勞。
晚上 voco 頂樓泳池/酒吧放鬆。
4/4 上午 voco Check-out (10:30) → Grab 前往 Four Points **寄放大型行李** → 轉車前往會安。 午餐：會安古鎮周邊高樓麵 (Cao Lầu) 或會安雞飯 重要移動日：輕裝前往會安。 Day 3 安排較輕鬆，平衡 Day 2 體力消耗。
下午/晚 La Charm Hoi An Check-in → **會安古鎮 (Hội An)** 漫步、參觀、逛夜市。 晚餐：會安河畔餐廳或在地特色小吃。 體驗會安著名的燈籠夜景與放水燈。
晚上
4/5 上午 La Charm 早餐 → **山茶半島 (Son Tra Peninsula)**：靈姑寺 (Linh Ung Pagoda) → Four Points Check-in。 午餐：搭車返回峴港 (約 12:00 抵達) → 於 Four Points 周邊輕食。
下午 峴港粉紅大教堂 → 龍橋附近用餐。 晚餐：峴港市區特色美食 (如：Mì Quảng) 或 Pizza 4P's。
晚上 SPA 按摩 (Queen Spa、Trees Spa 等) 或 Four Points 頂樓泳池/酒吧放鬆。
深夜 龍橋噴火表演 (約晚上9點 - 週末限定)
4/6 上午 **峴港市區探索：** 韓市場 (Han Market) 採買伴手禮 → 峴港大教堂 (粉紅教堂) 拍照。 午餐：韓市場周邊或當地市場 (Cồn Market) 體驗在地市場小吃。 市場購物請注意物品安全。
下午/晚 SPA 按摩或海灘放鬆。 晚餐：前往龍橋 (Dragon Bridge) 附近用餐或海鮮大排檔。 建議提前預約 SPA，今日非週末，無龍橋噴火秀。
深夜 Four Points 飯店設施深度放鬆，整理戰利品。
4/7 上午 悠閒享用 Four Points 早餐 → 整理行李、最後採買或沙灘散步。 午餐：於飯店附近或喜歡的餐廳。 預留充足時間整理行李。
16:00 從飯店出發前往峴港國際機場 (DAD)。 班機預計 18:45 起飛。 請至少在起飛前 2 小時抵達機場。

🗺️ 每日時間軸與路線詳情

Day 1 (4/2) - 入住與海灘散步 (建議停留：2.5hrs)

機場 (DAD) → **Grab (約 20min)** → voco Ma Belle Danang (Check-in) → 美溪海灘 (My Khe Beach) 散步。

Day 2 (4/3) - 文化探索：五行山 (建議停留：3hrs)

voco → **Grab (約 20min)** → **五行山 (Marble Mountains) 徒步遊覽** → 午餐 (周邊/途中) → **Grab** → voco 頂樓泳池/海鮮晚餐。

Day 3 (4/4) - 城市移動日：轉往會安 (建議停留：4hrs)

voco (Check-out) → **Grab (約 5min)** → Four Points by Sheraton Danang (寄放行李) → **Grab (約 50min)** → La Charm Hoi An Hotel & Spa (Check-in) → **步行/單車** → **會安古鎮 (Hội An)** (看燈籠夜景)。

Day 4 (4/5) - 告別會安與週末龍橋 (建議停留：5hrs)

La Charm (Check-out) → **Grab (約 1hr)** → **山茶半島 (Son Tra Peninsula) - 靈姑寺** → Four Points (Check-in & 取行李) → **步行** → 峴港粉紅大教堂 → 韓市場/崑市場。

**晚上：** 龍橋噴火表演 (週末限定，約 21:00) → SPA 按摩。

Day 5 (4/6) - 歷史文化與購物放鬆 (建議停留：4hrs)

Four Points → **Grab (約 15min)** → **佔婆博物館 (Cham Museum)** → 午餐 → **Grab/步行** → 購物中心 (Vincom Plaza 或 Lotte Mart) → 晚餐 → 第二次 SPA 或飯店 Lounge 喝酒。

Day 6 (4/7) - 悠閒告別 (建議停留：2hrs)

Four Points 頂層泳池悠閒時光 → 最後採買 → **16:00** → **Grab (約 20min)** → 峴港國際機場 (DAD)。

Day 1 (4/2) - 入住與海灘散步 (建議停留：2hrs)

機場 (DAD) → **Grab (約 20min)** → voco Ma Belle Danang (Check-in) → 美溪海灘 (My Khe Beach) 輕鬆晚餐。

Day 2 (4/3) - 體力挑戰：巴拿山 (建議停留：8hrs)

voco (8:00 前出發) → **Grab (約 1hr)** → **巴拿山 (Ba Na Hills)** (纜車、金橋、法國村、夢想樂園) → **Grab** (約 1hr) → 市區海鮮自助晚餐。

Day 3 (4/4) - 城市移動日：轉往會安 (建議停留：4hrs)

voco (Check-out) → **Grab (約 5min)** → Four Points (寄放行李) → **Grab (約 50min)** → La Charm Hoi An (Check-in) → **步行** → **會安古鎮 (Hội An)** (看燈籠夜景)。

Day 4 (4/5) - 告別會安與週末龍橋 (建議停留：5hrs)

La Charm (Check-out) → **Grab (約 1hr)** → **山茶半島 (Son Tra Peninsula) - 靈姑寺** → Four Points (Check-in & 取行李) → **步行** → 峴港粉紅大教堂 → 龍橋附近晚餐。

**晚上：** 龍橋噴火表演 (週末限定，約 21:00) → SPA 按摩。

Day 6 (4/7) - 悠閒告別 (建議停留：2hrs)

Four Points 頂層泳池悠閒時光 → 最後採買 → **16:00** → **Grab (約 20min)** → 峴港國際機場 (DAD)。

✅ 行前準備與注意事項

  • **簽證/護照：** 確認護照效期 (需六個月以上)。
  • **貨幣兌換：** 越南盾 (VND)。建議在機場或峴港市區金店換匯，匯率較佳。
  • **通訊：** 抵達機場後立即購買 **SIM 卡**或使用 eSIM，確保全程網路順暢。
  • **必備 Apps：** **Grab App** (叫車必備)、Google Maps、翻譯 App。
  • **SPA 預約：** 由於 Day 5/6 規劃了 SPA，建議提前在網路上預約 Queen Spa 或 Trees Spa 等熱門店家。
